# lets-go-out

## Problem
Many people want to attend everyday activities like movies, coffee, or sports but hesitate to go alone or lack companions with similar interests.

## Solution
lets-go-out is a platform where users can host an outing or activity and others can express interest to join, enabling simple and safe social connections around shared activities.

## Core Features (MVP)
- Host an outing (movie, coffee, sports, etc.)
- Discover available outings
- Send an interest request to join an outing
- Host can accept or reject requests

## Out of Scope (for MVP)
- Chat or messaging
- Dating features
- Payments
- Social feeds
- Advanced profiles

## Product Philosophy
- Simple by design
- Safety and user control first
- Focused on intent, not endless interaction
